# Crypto Pulse News AI

Интеллектуальный агрегатор крипто-новостей с автоматическим переводом на русский язык и интеграцией с Telegram-каналом.

## Основные функции:
* **Real-time мониторинг**: Получение свежих новостей через CryptoCompare API.
* **Smart Translation**: Автоматический перевод заголовков и описаний.
* **Telegram Integration**: Возможность быстрой публикации новостей в канал прямо из интерфейса админа.
* **Smart Images**: Динамический подбор релевантных изображений для каждой новости.

## Технологии:
* **Framework**: Next.js 16 (App Router)
* **Styling**: Tailwind CSS
* **Deployment**: Vercel
* **API**: CryptoCompare, Unsplash Source
